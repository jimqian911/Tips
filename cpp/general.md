# Examples 
* Example code. See [here][a] or [here](b)
```
#include<stdio.h>
int main( void )
{
    printf( "Hello World!\n" );
    return 0;
}
```
* Compile
```
$ g++ -o helloworld main.cpp
```
* Execute
```
$ ./helloworld
```
`Hello, world!` should appear.
